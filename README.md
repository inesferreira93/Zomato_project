#### source venv/bin/activate
### no linux / macOs:
#### source venv/bin/activate

#### to Run the tests just need to run: 
##### ```pytest``` OR ```pytest -s``` (if you want to see the details) OR ```pytest -k <name_of_file>```, such as: ```test_csv_import_to_db.py```
